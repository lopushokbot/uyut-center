# Гостиница «Уют» ★★★

Современный сайт трёхзвёздочной гостиницы «Уют» в г. Клинцы.

## Стек

- **React 18** + **Vite 6**
- CSS-in-JS (встроенные стили)
- Шрифты: Cormorant Garamond + Raleway (Google Fonts)

## Запуск

```bash
npm install
npm run dev
```

Сайт откроется на `http://localhost:3000`

## Сборка для продакшена

```bash
npm run build
npm run preview
```

Готовые файлы будут в папке `app/`.

## Интеграции (требуют настройки)

| Интеграция | Статус | Что нужно |
|---|---|---|
| Контур.Отель | 🟡 Заглушка | API-ключ и endpoint |
| Яндекс.Карты | 🟢 Работает | Виджет встроен |
| Виджет отзывов | 🟡 Внутренний | API Яндекс/Google/TripAdvisor |
| Онлайн-оплата | 🟡 UI готов | SDK платёжного шлюза (ЮKassa / Тинькофф) |

## Структура

```
├── index.html          # Точка входа
├── package.json
├── vite.config.js
├── app/                # Результат production-сборки
└── src/

    └── App.jsx         # Основной компонент сайта
```
