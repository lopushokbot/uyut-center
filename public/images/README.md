# Images Structure

## Rooms

Store room photos in dedicated folders:

- `rooms/single/`
- `rooms/comfort/`
- `rooms/business/`
- `rooms/luxe/`
- `rooms/junior-suite/`
- `rooms/triple/`
- `rooms/half-luxe/`

Recommended naming:

- `01.jpg` — main cover photo
- `02.jpg`, `03.jpg`, `04.jpg` — additional gallery photos

Update `gallery` arrays in `src/app/prices_and_rooms/page.jsx` when adding new room photos.

## Other sections

Suggested directories:

- `about/`
- `services/`
- `offers/`
- `restaurant/`
- `contact/`
